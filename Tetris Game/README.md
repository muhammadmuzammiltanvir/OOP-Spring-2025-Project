# OOP Tetris Game Project

## Description
This project is a Tetris game developed as part of the Object Oriented Programming (OOP) course. 
It is written in C++ and uses the SFML for graphics and sound.

The main objective of this project is to understand and implement core OOP concepts such as Abstraction, Encapsulation, Inheritance, and Polymorphism.

## Features
- Different colored blocks are used in the game  
- Boxes fall in a random order  
- Movement of blocks in different directions  
- Rotation functionality for blocks  
- Sound effects for:
  - Block movement  
  - Block rotation  
  - Game over  
- High score system using file handling  
  - High score is stored in `highscore.txt`  
  - Option to reset the high score  

## Controls
- **Up Arrow Key** : Rotate block  
- **Down Arrow Key** : Move block down  
- **Left Arrow Key** : Move block left  
- **Right Arrow Key** : Move block right  
- **Enter Key** : Start the game again after Game Over  

## Requirements
- C++ Compiler  
- SFML Library (not included in this repository)

## Note
Make sure SFML is properly installed and configured before compiling and running the project.
